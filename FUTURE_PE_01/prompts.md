# Prompt Engineering Documentation

## Prompt 1

Act as a professional website copywriter.

Create homepage copy for a beauty salon called "Glow & Grace Beauty Salon" located in Mumbai.

Requirements:
- Write a compelling hero section
- Create an About Us section
- Highlight key services
- Include trust-building content
- End with a strong call-to-action

Tone:
Friendly, professional, elegant, and persuasive.

## Prompt 2

Create detailed website content for the following salon services:

1. Haircuts
2. Hair Coloring
3. Bridal Makeup
4. Facial Treatments
5. Nail Art

Requirements:
- 100-150 words per service
- Benefits-focused
- Client-friendly language

## Prompt 3

Generate 5 website CTA sections for a beauty salon.

Examples:
- Book Appointment
- Contact Us
- Free Consultation

Tone:
Friendly and conversion-focused.
